# **App Name**: BudgetWise

## Core Features:

- Income Tracking: Log and categorize income sources.
- Expense Tracking: Record and categorize expenses with optional descriptions and tags.
- Budget Creation: Set monthly budgets for different categories, allowing users to allocate funds effectively.
- Spending Insights: Provide a summary of spending patterns using charts, such as bar graphs for comparing spending across categories or line graphs to show spending trends.
- Dark and Light Mode: Option to switch between dark and light mode.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence.
- Background color: Light gray (#F0F2F5) for a clean, readable interface.
- Accent color: Soft teal (#4CAF50) to indicate positive changes or financial improvements.
- Body and headline font: 'Inter', a sans-serif font with a modern look suitable for a financial application.
- Use simple, professional line icons representing each expense category.
- Ensure a clean, intuitive layout for clear navigation and data visualization.
- Use subtle transitions and animations to enhance user experience when displaying insights.
